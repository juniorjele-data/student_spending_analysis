
# Student Spending Analysis & Prediction

## 📊 Overview
This project analyzes student spending behavior across different categories such as food, housing, and technology. It also explores whether income affects spending using a machine learning model.

---

## 🔍 Key Insights
- Tuition is the largest expense among students
- Income has little impact on food spending
- Seniors spend more on food than juniors
- Biology students spend more on technology than Computer Science students

---

## 🤖 Machine Learning
A Linear Regression model was used to predict food spending based on monthly income.

Result:
- Weak relationship between income and food spending
- Model shows nearly flat trend
